To run the project:
    1. Open the 'runOptimization.ipynb' file and select a Python interpreter.
    2. Make sure all imported libraries are installed
    3. Click "Run All" to run each cell, which will produce a full optimization with default parameters.
    4. To change model parameters, scroll down to the cell for the two respective TSP problems (ATSP and STSP) 
        and edit the model parameters.
    5. Simple run statistcs are displayed at the bottom of the notebook.

To view documentation for the project:
    1. Open the folder titled 'dOxygen'
    2. Double click the file 'index.html' or open it with a web browser
    3. Navigate to the classes section to see the tspPSO file and its classes.
